description: List the CaBrain brains this token can reach, with memory counts.
---

Call the CaBrain **brain_list** MCP tool and present the brains as a short table: namespace, memory count, last-updated.

If the user passed a brain name in `$ARGUMENTS`, instead call **brain_details** for that namespace and summarize it: memory count, breakdown by type/source, open knowledge gaps, and recent recall activity.
